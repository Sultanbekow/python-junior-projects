# 🌤️ Projekt 2: Wetter-App für iPhone

Läuft mit **Pythonista 3** auf dem iPhone.

## Installation
1. Pythonista 3 aus dem App Store installieren (ca. 15€)
2. API-Key holen: https://home.openweathermap.org/users/sign_up
3. In `weather_app.py` den API_KEY eintragen
4. Datei auf iPhone übertragen (AirDrop, iCloud, etc.)
5. In Pythonista 3 öffnen und auf Run tippen

## Features
- 📱 Native iOS-Optik
- 🔍 Städtesuche
- 🌡️ Echtzeit-Wetterdaten (Celsius, Wind, Luftfeuchtigkeit)
